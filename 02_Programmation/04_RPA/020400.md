À qui s’adresse cette formation ?

- **Pour qui** : Développeurs, testeurs, consultants, chefs de projet
- **Objectif** : Automatiser des tâches répétitives avec Robot Framework
- **Niveau** : Débutant à intermédiaire
- **Prérequis** : Bases en programmation et manipulation d’applications

## Robot Framework dans ce cours

*   Utilisé comme **moteur d’automatisation RPA**
*   Permet d’enchaîner des actions sur plusieurs systèmes
*   Sert de support pédagogique pour comprendre les principes du RPA

> Ici, le RPA consiste à **automatiser des tâches**, pas à tester du code.

# Parcours de formation

## Jour 1 : Introduction et prise en main

### Présentation de Robot Framework

*   Qu’est-ce que Robot Framework
*   À quoi il sert concrètement
*   Cas d’usage principaux

### Installation de Robot Framework

*   Installation de l’outil
*   Vérification du bon fonctionnement
*   Premier test simple

### Méthodologie et syntaxe

*   Concepts fondamentaux
*   Structure d’un script Robot Framework
*   Variables, mots-clés, tests
*   Lecture des logs et rapports

### Atelier pratique

*   Installation sur les postes
*   Création de premiers scripts
*   Compréhension de l’exécution

## Jour 2 : Applications pratiques

### Application à la lecture de fichiers

*   Lecture et écriture de fichiers texte
*   Manipulation de fichiers CSV
*   Vérification automatique de contenu

**Cas réel**

*   Générer et contrôler un rapport automatiquement

### Application aux bases de données

*   Connexion à une base de données
*   Exécution de requêtes SQL
*   Vérification et insertion de données

**Cas réel**

*   Mettre à jour ou contrôler un référentiel utilisateurs

### Application aux sites web

*   Interaction avec une application web
*   Saisie de données
*   Vérification de messages ou de résultats

**Cas réel**

*   Automatiser une saisie ou un contrôle applicatif

```{admonition} Méthodes pédagogiques
- **Théorie courte** : comprendre l’essentiel
- **Pratique immédiate** : automatiser par l’exemple
- **Progressivité** : du simple au concret
```
