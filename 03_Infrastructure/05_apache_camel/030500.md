Apache Camel est un framework d'intégration open source basé sur les *Enterprise Integration Patterns* (EIP).
Il permet de connecter des systèmes hétérogènes via des routes déclaratives.

Ce module couvre l'installation d'Apache Camel et Karavan via Docker, puis la mise en œuvre d'une route de recopie de tables MS-SQL vers PostgreSQL.
